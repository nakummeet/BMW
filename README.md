# 🚗 BMW M Series Landing Page

A modern **BMW M Series inspired landing page** built with **HTML, CSS, and JavaScript**.  
The project focuses on creating a **premium automotive UI experience** with smooth scrolling, animations, and interactive elements.

⚠️ This project is created for **educational and portfolio purposes only** and is **not affiliated with BMW AG**.

---

## 🌐 Live Demo

https://bmw-swart.vercel.app/

---

## ✨ Features

- 🖥️ Modern Landing Page Design
- 🎬 Hero Section Animation
- 🧭 Smooth Scrolling (Locomotive Scroll)
- 🎯 Interactive Model Showcase
- 🖱️ Custom Cursor Animation
- 🖼️ Image Hover Reveal
- 🚗 Individual Model Pages
- 📱 Responsive Layout

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (ES6)

### Libraries
- GSAP (GreenSock Animation Platform)
- Locomotive Scroll
